Selectores

Estos selectores apuntan a elementos HTML específicos. Por ejemplo, el selector de etiqueta p apuntaría a todos los elementos

en la página

Selectores de clase

Los selectores de clase apuntan a elementos HTML que tienen una clase específica. Por ejemplo, el selector .mi-clase apuntaría a todos los elementos que tienen la clase mi-clase.

Learn More

Selectores de ID

Los selectores de ID apuntan a elementos HTML que tienen un ID específico. Por ejemplo, el selector #mi-id apuntaría al elemento que tiene el ID mi-id.
