Cómo Clonar un Repositorio Usando Git Bash

En esta guía te mostramos cómo clonar un repositorio de GitHub a tu máquina local usando Git Bash.

  1. Abre Git Bash en tu computadora.
  2. Dirígete al repositorio de GitHub que deseas clonar.
  3. Copia la URL del repositorio haciendo clic en el botón "Code" y seleccionando HTTPS.
  4. En Git Bash, ejecuta el siguiente comando:
    5. git clone URL-del-repositorio
  5. Reemplaza URL-del-repositorio con la URL que copiaste en el paso anterior.
  6. Presiona Enter para clonar el repositorio en tu carpeta actual.
