1. ¿Qué es JavaScript?

JavaScript es un lenguaje de programación que se utiliza para crear páginas web interactivas y dinámicas. Se ejecuta directamente en el navegador del usuario, sin necesidad de compilarse previamente.
Algunas de las características de JavaScript son:

  • Es un lenguaje interpretado, por lo que no necesita ser compilado antes de ejecutarse.
  • Es de tipado débil y dinámico, lo que significa que no es necesario declarar explícitamente el tipo a una variable.
  • Es un lenguaje basado en prototipos, lo que ofrece flexibilidad y capacidad de herencia dinámica.
  • Usa funciones de orden superior, lo que significa que las funciones pueden ser tratadas como cualquier otro valor.

    • Algunas de las cosas que se pueden hacer con JavaScript son:
  • Actualizar fuentes de redes sociales
  • Mostrar animaciones y mapas interactivos
  • Añadir juegos
  • Validar los valores de entrada de un formulario web
  • Transmitir información sobre los hábitos de lectura de los usuarios

    • JavaScript fue creado por Brendan Eich en 1995 y es una de las tres tecnologías web principales, junto con HTML y CSS. Para facilitar el trabajo de los desarrolladores, existen frameworks de JavaScript, como Ember, Singular, React, y Vue. js.

    2. Variables y Tipos de Datos

    Las variables en JavaScript almacenan valores que se pueden usar y modificar en el código. Variables Utiliza variables como nombres simbólicos para valores en tu aplicación. Los nombres de las variables, llamados identificadores, se ajustan a ciertas reglas. Un identificador de JavaScript debe comenzar con una letra, un guión bajo (_) o un signo de dólar ($). Los siguientes caracteres también pueden ser dígitos (0-9). Dado que JavaScript distingue entre mayúsculas y minúsculas, las letras incluyen los caracteres "A" a "Z" (mayúsculas), así como "a" a "z" (minúsculas). Puedes utilizar la mayoría de las letras ISO 8859-1 o Unicode como å y ü en los identificadores. (Para obtener más detalles, consulta esta publicación del blog). También puedes usar Secuencias de escape Unicode como caracteres en identificadores. Algunos ejemplos de nombres legales son Number_hits, temp99, $credit y _name.

    Declaración de variables
    Puedes declarar una variable de dos formas:

  • la palabra clave var. Por ejemplo, var x = 42. Esta sintaxis se puede utilizar para declarar variables locales y globales, dependiendo del contexto de ejecución.
  • Con la palabra clave const o let. Por ejemplo, let y = 13. Esta sintaxis se puede utilizar para declarar una variable local con ámbito de bloque. (Ve el Ámbito de variables abajo.)

    • También puedes simplemente asignar un valor a una variable. Por ejemplo, x = 42. Este formulario crea una variable global no declarada. También genera una advertencia estricta de JavaScript. Las variables globales no declaradas a menudo pueden provocar un comportamiento inesperado. Por lo tanto, se desaconseja utilizar variables globales no declaradas.

    Evaluar variables
    Una variable declarada usando la instrucción var o let sin un valor asignado especificado tiene el valor de undefined. Un intento de acceder a una variable no declarada da como resultado el disparo de una excepción ReferenceError


    Tipos de datos

    los tipos de datos son la naturaleza del contenido de una variable o constante. Por ejemplo, si tenemos una variable con contenido 44, su tipo de dato es un número. Si tenemos una variable con contenido "Manz", su tipo de dato es texto. En muchos lenguajes, el programador debe indicar el tipo de dato que tiene una constante o variable. Sin embargo, Javascript nos da la libertad de no tener que indicarlo. Sin embargo, recuerda que aunque no lo indiquemos, Javascript le asigna un tipo de dato dependiendo del contenido.

    Tipos de datos primitivos

  • Number: Valor numérico (enteros, decimales...)
  • BigInt: Valor numérico muy grande Ver BigInt
  • String: Valor de texto (cadenas de texto, carácteres...)
  • Boolean: Valor booleano (valores verdadero o falso)
  • Symbol: Símbolo (valor único)
  • undefined: Valor sin definir (variable sin inicializar)

    • Tipos de datos no primitivos

  • Object Objeto (estructura más compleja)
  • Function: Función (función guardada en una variable)

  • null. Representa un valor vacío o ausencia de información.

    • 3. Funciones

    Una función es un bloque de código que realiza una tarea específica. Las funciones nos permiten agrupar líneas de código en tareas con un nombre, para que, posteriormente, podamos hacer referencia a ese nombre para realizar todo lo que se agrupe en dicha tarea. Para usar funciones hay que hacer 2 cosas:

    1. Declarar la función: Preparar la función, darle un nombre y decirle las tareas que realizará.
    2. Ejecutar la función: «Llamar» a la función para que realice las tareas de su contenido.

    Declaración
    En el siguiente ejemplo veremos la declaración de una función llamada saludar:

    • // Declaración de la función "saludar"
    • function saludar() {
    • // Contenido de la función
    • console.log("Hola, soy una función");
    • }

    El contenido de la función es una línea que mostrará por consola un saludo. Sin embargo, si escribimos estas 4-5 líneas de código en nuestro programa, no mostrará nada por pantalla. Esto ocurre así porque solo hemos declarado la función (le hemos dicho que existe), pero aún nos falta el segundo paso, ejecutarla, que es realmente cuando se realizan las tareas de su contenido.

    Ejecución
    Veamos, ahora sí, el ejemplo completo con declaración y ejecución:

    • // Declaración de la función "saludar"
    • function saludar() {
    • // Contenido de la función
    • console.log("Hola, soy una función");
    • }
    • // Ejecución de la función
    • saludar();

    En este ejemplo hemos declarado la función y además, hemos ejecutado la función (en la última línea) llamándola por su nombre y seguida de ambos paréntesis, que nos indican que es una función. En este ejemplo, si se nos mostraría en la consola Javascript el mensaje de saludo.

    4. Eventos

    Los eventos en JavaScript permiten responder a las interacciones del usuario, como hacer clic en un botón.
    Eventos del mouse: Son acciones que ocurren cuando el usuario interactúa con un dispositivo señalador, como un mouse. Algunos ejemplos de eventos del mouse son:

  • click: Se activa cuando el mouse hace clic en un elemento
  • dblclick: Se activa cuando el mouse hace clic dos veces en un elemento
  • contextmenu: Se activa cuando el mouse hace clic derecho en un elemento
  • mouseover: Se activa cuando el cursor del mouse ingresa a un elemento
  • mouseout: Se activa cuando el cursor del mouse abandona un elemento
  • mousedown: Se activa cuando el botón del mouse se presiona en un elemento
  • mouseup: Se activa cuando el botón del mouse se suelta en un elemento

    • Eventos del teclado: Algunos ejemplos de eventos del teclado son:
  • keydown
  • keypress
  • keyup

    • Eventos de formularios: Algunos ejemplos de eventos de formularios son:
  • focus
  • blur
  • change
  • submit

    • Eventos de la ventana: Algunos ejemplos de eventos de la ventana son:
  • scroll
  • resize
  • hashchange
  • load
  • unload
