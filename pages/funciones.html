Funciones en JavaScript

Introducción

En JavaScript, las funciones son bloques de código diseñados para realizar una tarea específica. Permiten organizar y reutilizar el código, haciendo que el desarrollo sea más eficiente y claro. Las funciones pueden aceptar parámetros y devolver valores, y son uno de los conceptos fundamentales en JavaScript y en la programación en general.

¿Qué es una función?

Una función en JavaScript es un conjunto de instrucciones agrupadas bajo un nombre, que se ejecutan cuando se llama a esa función. Las funciones ayudan a evitar la repetición de código y facilitan el mantenimiento.

Sintaxis Básica

La sintaxis básica de una función en JavaScript incluye la palabra clave function, un nombre para la función, paréntesis () y un bloque de código entre llaves {}. Así:
function nombreFuncion() {
// código a ejecutar
}

Tipos de Funciones en JavaScript

  • Funciones Declarativas
  • Funciones Expresadas (Funciones Anónimas)
  • Funciones Flecha (Arrow Functions)
  • Parámetros y Argumentos
  • Valor de Retorno
  • Funciones de Orden Superior
  • Funciones Asíncronas

Funciones Declarativas

Son funciones definidas con la palabra clave function, seguida del nombre de la función y el bloque de instrucciones. Estas funciones pueden ser llamadas antes de su declaración en el código.
function saludar() {
console.log("¡Hola!");
}

Funciones Expresadas (Funciones Anónimas)

Las funciones expresadas se asignan a una variable y pueden o no tener un nombre. Solo se pueden llamar después de su definición, ya que no son elevadas (hoisted) como las funciones declarativas.
const saludar = function() {
console.log("¡Hola!");
};

Funciones Flecha (Arrow Functions)

Introducidas en ES6, las funciones flecha tienen una sintaxis más corta y no crean su propio contexto this. Son ideales para funciones de una sola línea o de callback.
const saludar = () => console.log("¡Hola!");

Parámetros y Argumentos

Las funciones pueden recibir valores como entrada, llamados "parámetros", que son utilizados dentro de la función. Los valores que se pasan cuando se llama a la función se denominan "argumentos".
function sumar(a, b) {
return a + b;
}

Valor de Retorno

Una función puede devolver un valor utilizando la palabra clave return. Esto permite que otras partes del código utilicen el valor generado por la función.
function multiplicar(a, b) {
return a * b;
}

Funciones de Orden Superior

Las funciones de orden superior son aquellas que pueden aceptar otras funciones como argumentos o devolver una función como resultado. Son comunes en operaciones de arreglos, como map, filter y reduce.
const duplicar = (n) => n * 2;
const numeros = [1, 2, 3];
const resultado = numeros.map(duplicar); // [2, 4, 6]

Funciones Asíncronas

Las funciones asíncronas permiten manejar operaciones que toman tiempo, como solicitudes a servidores. JavaScript utiliza async y await para escribir código asíncrono de manera más sencilla y legible.
async function obtenerDatos() {
const respuesta = await fetch("https://api.example.com/data");
const datos = await respuesta.json();
return datos;
}
